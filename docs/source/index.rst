pyonedrive
==========

This is the API documentation for ``pyonedrive``, a OneDrive command
line client (and at the same time, an API client) using `OneDrive API
v1.0 <https://dev.onedrive.com/README.htm>`_. For introduction,
installation, and usage instructions, see `README.rst
<https://github.com/zmwangx/pyonedrive>`_ on GitHub.
